Fixing YouTube bugs in the new GUI (missing blur and rounded corners of button and window backgrounds), just add [these](https://github.com/Dim0s/YouTube-fix-GUI/blob/main/filters/fix-new-GUI.txt) filters to uBlock Origin, or install [this](https://github.com/Dim0s/YouTube-fix-GUI/raw/refs/heads/main/extensions/YouTube%20fix%20new%20GUI.zip) browser extension.

You can restore the old GUI using [this](https://github.com/Dim0s/YouTube-fix-GUI/blob/main/filters/restore-old-GUI.txt) filters to uBlock Origin, or install [this](https://github.com/Dim0s/YouTube-fix-GUI/raw/refs/heads/main/extensions/YouTube%20restore%20old%20GUI.zip) browser extension, see the demo at the end of the page.

extension install:

download and unpack it, in the browser in the extensions tab, enable "Developer Mode" and click on "Load unpacked extension", specify the path to the downloaded and unpacked folder, after installation, the extension is automatically enabled only for the YouTube site

extension includes fix from [this repo](https://github.com/Dim0s/YouTube-fix-control)
___
исправление багов на ютубе в новом гуи (недостающие размытие и закругление углов фона кнопок и окон), добавьте [эти](https://github.com/Dim0s/YouTube-fix-GUI/blob/main/filters/fix-new-GUI.txt) фильтры в uBlock Origin, или установите [это](https://github.com/Dim0s/YouTube-fix-GUI/raw/refs/heads/main/extensions/YouTube%20fix%20new%20GUI.zip) расширение для браузера

восстановить старый гуи можно [этим](https://github.com/Dim0s/YouTube-fix-GUI/blob/main/filters/restore-old-GUI.txt) фильтром для uBlock Origin, или [этим](https://github.com/Dim0s/YouTube-fix-GUI/raw/refs/heads/main/extensions/YouTube%20restore%20old%20GUI.zip) расширением для браузера, демо смотреть в конце страницы.


установка расширения:

скачайте и распакуйте, в браузере во вкладке расширений включите "Режим разработчика" и нажмите на "Загрузить распакованное расширение", укажите путь к загруженной и распакованной папке, после установки расширение автоматом включается только для сайта ютуба.

в расширение включены исправления из [этой репы](https://github.com/Dim0s/YouTube-fix-control)
